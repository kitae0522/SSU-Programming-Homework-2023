# Sum of Prime Numbers

100보다 작은 자연수 n을 입력 받아서 1부터 n사이의 모든 소수의 합을 출력하세요.

## 입출력 예시
(입력 #1)
```
12
```
(출력 #1)
```
28
```

(입력 #2)
```
30
```
(출력 #2)
```
129
```

## 채점 방법

다음의 명령어를 실행하여 채점을 진행합니다.

```Makefile
make clean all test
```
