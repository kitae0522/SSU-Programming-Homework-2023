# n-Hello Git!

Clone받은 내용을 바탕으로 코드를 수정하여 사용자에게 입력받은 숫자만큼 반복해서 한 줄에 하나씩 "Hello Git!"이 출력되도록 작성해주세요.

## 입출력 예시
(입력)
```
3
```

(출력)
```
Hello Git!
Hello Git!
Hello Git!
```

## 채점 방법

다음의 명령어를 실행하여 채점을 진행합니다.

```Makefile
make clean all test
```

makefile의 내용이 올바르게 설정되어 있는지 확인해주세요.
