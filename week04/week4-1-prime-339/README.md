# Prime 339

사용자로부터 자연수를 하나 입력받고, 그 숫자보다 작거나 같은 소수들 중 숫자 내에 "339"가 포함된 것이 모두 몇 개인지 출력하세요.

- 소수: 숫자 자신과 1로만 나눠 떨어지는 수
- "339"가 들어간 숫자의 예시: 339, 1339, ..., 9339, ... 33900, ...
- 입력으로는 100,000 이하의 자연수만 들어간다고 가정하세요.
- 결과는 숫자만 출력하세요.


## 입출력 예시
(입력 #1)
```
3000
```
(출력 #1)
```
1
```
(입력 #2)
```
30000
```
(출력 #2)
```
12
```
